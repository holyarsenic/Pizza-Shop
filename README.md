# 🍕 Pizza Shop

This project is a simple and responsive Pizza Shop website created using HTML, CSS, and JavaScript. It includes smooth animations, a modern layout, and interactive elements.

## Features

* Responsive navigation
* Animated sections using GSAP
* Custom cursor effect
* Login popup interface
* Menu and contact sections

## Built With

* HTML
* CSS
* JavaScript
* GSAP
